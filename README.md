# 🚆 Online Reservation System

## 📌 Oasis Infobyte Java Development Internship

### Task 1 - Online Reservation System

## Project Description

The Online Reservation System is a Java Swing desktop application that allows users to reserve railway tickets, search reservations using PNR, cancel tickets, and view all reservations. The application uses MySQL as the backend database and JDBC for database connectivity.

---

## Features

- User Login
- Reserve Train Ticket
- Fetch Train Details
- Search Reservation by PNR
- View All Reservations
- Cancel Reservation
- Input Validation
- MySQL Database Integration

---
